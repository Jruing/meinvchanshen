# 美女缠身

经典河南扑克牌游戏，多人聚会喝酒助兴必备。纯前端单文件应用，无任何依赖。

## 游戏设置

- **大小王**：可选择保留或移除。保留时大小王作为万能牌，与任意牌配对均判定为 4 杯
- **癞子**：可从 A～K 中指定一个牌面作为癞子（万能牌），效果与大小王一致，与任意牌配对均判定为 4 杯。也可以不设置
- **玩家**：输入至少 2 名玩家方可开始游戏

## 游戏玩法

1. 完成设置后点击"开始游戏"
2. 开局自动在牌桌上翻开 **玩家人数 + 1** 张底牌
3. 玩家轮流从牌堆中点击**揭牌**（抽一张牌，牌面朝下显示"?"）
4. 点击牌桌上的任意一张底牌进行**放置**
5. 系统自动判定揭开的牌与目标底牌的关系，弹窗显示喝酒杯数或过关
6. 放置后，揭开的牌替换目标底牌成为新的底牌，轮到下一位玩家
7. 牌堆发完则游戏结束

## 喝酒规则

| 关系 | 杯数 |
|------|------|
| 相同数字 / 大小王配对 / 癞子配对 | 4 杯 |
| 相同花色 + 顺子 | 3 杯 |
| 相同花色、不同数字 | 2 杯 |
| 不同花色 + 顺子 | 1 杯 |
| 无关系 | 过关（0 杯） |

**顺子定义：** 两张牌数字相差 1 或 2（例如 4 和 5、4 和 6）。K 和 A 之间也构成顺子（相差 12 视为相邻）。

## 界面功能

- **牌桌区域**：展示当前底牌，放置阶段底牌高亮提示可点击
- **牌堆**：显示剩余牌数，点击揭牌
- **喝酒/过关弹窗**：每轮结束弹窗展示判定结果及配对的两张牌
- **玩家排名**：按喝酒总杯数降序排列，最多的玩家标记皇冠
- **游戏统计**：实时显示总喝酒数和当前轮数
- **历史记录**：展示最近 15 条操作记录

## 运行方式

浏览器直接打开 `index.html` 即可，无需安装任何依赖。
